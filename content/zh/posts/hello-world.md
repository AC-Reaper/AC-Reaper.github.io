很久没有正经写点东西了。

今天搭了这个博客，没用任何花哨的服务，就是 Hugo + GitHub Pages，一个最朴素的组合。
图省事的话，我只需要写一个 Markdown 文件，`git push` 一下，几十秒后内容就出现在公网上。

## 为什么又开一个博客？

这些年存的笔记越来越多，但大部分都散在备忘录、Notion、Obsidian、各种聊天工具里，
真到想回头找的时候，反而一片模糊。我想试试看，把一部分内容公开记下来 —— 不是写给谁看，
更多是写给未来的自己。

## 大概会写些什么？

- **技术**：踩过的坑、用得顺手的工具、半夜灵光乍现的小项目
- **生活**：旅行的片段、看过的书和电影、最近在听的歌
- **碎碎念**：偶尔的胡思乱想

不会强迫自己定期更新。**有想说的就写，没有就不写**，这是我给自己定的唯一规则。

## 写在最前面

如果你恰好路过、看到了这里，欢迎留下你的足迹。可以通过 GitHub 找到我。

那么，开始吧。
